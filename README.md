# YT-ScamDatabase
Welcome to the database containing many scam channels and videos on youtube! The data shown is collecting from my very own Scam Scanner bot on my [Discord Sever - Scam Finder](https://discord.gg/fpJbN4kBEk). The bot uses specific search queries using the Youtube API on Google Cloud to find scam videos and channels. This repository serves as a hub to list these results.
